Configuration
-------------
1. Google Mail username and password needs to be configured in application.properties.
2. To-email is to be changed in EmailTest.java; String constant TO_EMAIL. 
3. Configure access for sending email through Google SMTP at https://www.google.com/settings/security/lesssecureapps

More
----
1. For a glance of ways of sending email through Google SMTP, visit https://support.google.com/a/answer/176600
