JSON-LD
=======

Hypermedia serialization with JSON-LD.

### Requirements

- Maven
- JDK 8
- JSON-LD

### Running
To build and start the server simply type

```bash
$ mvn clean install
$ mvn spring-boot:run
```

Now with default configurations it will be available at: [http://localhost:8080](http://localhost:8080)

Enjoy it :)
