Manual tests

The following are tests which are not covered by integration tests, but that can be run manually in order to check the functionality, either in the browser or in the logs, depending on the case.

  • File Upload
  • User POJO Post
  • Webhook narrowed to the '/user/*' URL
  • Load Configuration in a Route
  • Trigger internal server error (and test the custom error 500 page)
  • Trigger BaeldungException (and test the Global Exception Handler)
