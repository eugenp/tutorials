### Build the Project
```
mvn clean install
```

### Run the Project

- Run the application using Maven Cargo plugin.
```
mvn cargo:run
```
- Go to the login page at http://localhost:8082/spring-security-web-login/login.html
- Login using ```user1/user1Pass``` details.
